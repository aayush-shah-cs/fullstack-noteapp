# NexNote

NexNote is a modern full-stack note-taking application built using React, Express, and MongoDB.  
It allows users to create, organize, archive, and manage notes with a clean and responsive interface.

---

## 🚀 Features

- Create Notes
- Edit Notes
- Delete Notes
- Archive Notes
- Trash Notes
- Folder Management
- Calendar View
- Color Filtering
- Persistent MongoDB Storage
- Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
